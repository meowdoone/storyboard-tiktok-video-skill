# 故事版的 TikTok 视频 Skill

一个面向商品短视频制作的 Codex skill：把产品链接、产品图、截图或简短产品描述，整理成 TikTok/TK 可用的视频故事版包。

默认只输出三项：

1. 产品图
2. 故事版
3. 15 秒或者 30 秒视频脚本文字

适合 FYPro、StoreClaw、图生视频、TikTok creator demo、UGC 广告前期分镜。

关键规则：产品图是 source of truth。故事版必须严格延续产品外观；脚本文字必须严格对应故事版。

## 为什么做这个 skill

很多商品视频任务会过早进入关键词、脚本、卖点分析，导致产品真实图和画面节奏不清楚。这个 skill 强制先看产品，再看故事版，最后看 15 秒脚本，让视频生成和人工沟通都更清晰。

## 使用方式

把 `SKILL.md` 安装到 Codex skills 目录后，可以这样使用：

```text
用故事版的 TikTok 视频 skill，帮我基于这个产品链接输出：产品图、故事版、15 秒视频脚本文字。
```

```text
用故事版的 TikTok 视频 skill，帮我基于这个产品图输出：产品图、故事版、30 秒视频脚本文字。
```

多产品：

```text
从这个品牌里找三个适合 TikTok 的产品，按产品图 -> 故事版 -> 15 秒视频脚本文字输出。
```

换图：

```text
这个故事版不够好，换一个图，但保持产品图、故事版、15 秒脚本文字三个部分。
```

关键词：

```text
基于这张故事版写 FYPro 关键词，必须保持故事版里的同一个人物、同一套衣服、同一间厨房、同一个产品。
```

## 默认格式

```markdown
## 产品名称

### 1. 产品图

### 2. 故事版

### 3. 15 秒或者 30 秒视频脚本文字
```

脚本表：

```markdown
| 时间 | 画面 | 屏幕字幕 | 旁白 / 口播 |
|---|---|---|---|
| 0-2s | ... | ... | "..." |
| 2-5s | ... | ... | "..." |
| 5-8s | ... | ... | "..." |
| 8-11s | ... | ... | "..." |
| 11-15s | ... | ... | "..." |
```

## 设计原则

- 产品图先行，不让故事版脱离产品真实形态。
- 故事版默认是 5 格编号竖图，每格都是 9:16 TikTok 关键帧。
- 故事版必须是连续产品使用链路：hook -> 产品进入 -> 产品过程/细节 -> 结果/使用 -> payoff。
- 5 格故事版可以对应 15 秒，也可以支撑 30 秒脚本。
- TikTok UGC 感优先，不做棚拍广告片。
- 真实手机拍摄、自然光、普通生活场景。
- 默认不输出 FYPro 关键词，除非用户另问。
- 如果用户另问关键词，关键词必须以“use the attached storyboard/reference frame”为前提，不允许模型重新生一个人物或场景。
